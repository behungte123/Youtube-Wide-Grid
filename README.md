
# 🎬 Youtube Wide Grid

### Mở rộng layout YouTube — Xem nhiều video hơn, cuộn ít hơn.

Một Chrome Extension giúp tối ưu không gian hiển thị của YouTube bằng cách
tăng số lượng video trên mỗi hàng mà **không ảnh hưởng Shorts hoặc UI chính**.

---

## 📌 Giới thiệu

YouTube mặc định hiển thị thumbnail khá lớn khiến màn hình bị lãng phí,
đặc biệt với màn hình rộng hoặc độ phân giải cao.

**Youtube Wide Grid** giải quyết vấn đề đó bằng cách:

- Tăng số cột video
- Tối ưu khoảng cách grid
- Hiển thị nhiều nội dung hơn cùng lúc

👉 Trải nghiệm YouTube gọn gàng và hiệu quả hơn.

## ⚙️ Cài đặt

### Cài đặt thủ công (Developer Mode)

**Bước 1 — Clone repository**

```bash
git clone https://github.com/behungte123/Youtube-Wide-Grid.git
```
---

**Bước 2 — Mở trang Extension
```
chrome://extensions/
```
---

**Bước 3 — Load extension

👉 Load unpacked
👉 Chọn thư mục project
🎉 Extension sẵn sàng hoạt động!

🧠 Cách hoạt động
